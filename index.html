Bienvenidos a nuestro portafolio

Este sitio web presenta de manera general a los integrantes del grupo de Tecnologías Computacionales I de la carrera de Ingeniería en Computación de la Facultad de Ingeniería de la UAEMex.

Aquí se muestran nuestros perfiles, habilidades técnicas, formación académica, proyectos y áreas de interés, con el propósito de reflejar nuestro desarrollo académico y profesional.

Sobre el grupo

Somos estudiantes de Ingeniería en Computación con interés en distintas áreas de la tecnología, entre ellas el desarrollo web, la programación, las bases de datos, las redes, el soporte técnico y el uso de herramientas colaborativas para la construcción de proyectos.

Objetivo del portafolio

El objetivo de este portafolio grupal es presentar de forma organizada los perfiles de los integrantes, así como sus habilidades, conocimientos, formación y proyectos, integrando en un solo espacio la información académica y técnica del equipo.

Áreas y habilidades generales

Desarrollo Web HTML CSS JavaScript Python Java Bases de Datos MySQL PostgreSQL Linux Git y GitHub Redes Soporte Técnico Mantenimiento de Equipos Trabajo en Equipo

Nuestro enfoque

Este portafolio no solo funciona como una presentación del grupo, sino también como una evidencia de nuestras competencias, intereses y evolución en el área de tecnologías computacionales.
